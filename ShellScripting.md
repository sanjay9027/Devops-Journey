# Shell Scripting

### Declare variable & print it

`#! /bin/bash 
NAME="Sanjay" 
echo "$NAME" 
echo $NAME  
echo "${NAME}" `

### Give Permission to access file
`chmod 777 <file_name>`

### Read data from UI & Print

`#!/bin/bash
echo "Enter your age ."
read AGE  
echo "My age is ${AGE}"`

`sleep 2  -> add 2 second sleep in execution`
